# Tmux Settings

My Tmux settings

## Usage

### Installation

1. Clone this repository.
   - `git clone <repository-url> <tmux-settings-path>`
1. Create a link to the file `.tmux.conf` in the home directory.
   - `ln -s <tmux-settings-path>/.tmux.conf ~/`

### Additional Setup

#### Install `xsel`

`xsel` is used to copy text to the clipboard. Installation may be needed if it
is not included by default.

1. Install `xsel`.
   - `sudo apt install xsel`
